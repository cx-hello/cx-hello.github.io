如何把要想保存的文章转为 Markdown 格式

[![](https://cdn3.jianshu.io/assets/default_avatar/3-9a2bcc21a5d89e21dafc73b39dc5f582.jpg)](/u/80f230349cb5)

前言

网上我们看到一些好文章的时候，总会想要把它转载过来，或者保存到自己的 Markdown 文档里，但是这时候就有一个问题，那就是网页上的文章都是 HTML 格式的，如何能够保存成 Markdown 格式呢？本文将告诉你一个简单的办法。

环境

  • Chrome 浏览器或其他可以审查元素的浏览器（可自行搜索不同浏览器如何审查元素）。
  • 在线 HTML 转 Markdown 工具，我使用的是 http://tool.lu/markdown/
    ![](//upload-images.jianshu.io/upload_images/5392482-35c838f7e51ff2b9.png)

步骤

以 阮一峰老师的 这篇文章为例，偷懒直接上图了，哈哈

  • 右键审查元素
  • 复制文章主体内容的 HTML 代码
  • 在转化工具中转化
    ![](//upload-images.jianshu.io/upload_images/5392482-79862445b6ad516d.gif)

然后再看看转化的 Markdown ，可能有的需要删除头和尾的 HTML 代码，剩下的就是华丽丽的 Markdown 格式的内容了。

Enjoy it !

如果觉得文章对你有用，可以请我喝杯咖啡~
